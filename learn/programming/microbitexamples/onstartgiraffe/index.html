Getting started with the micro:bit

This tutorial will talk about the micro:bit, how to write your first program for it, and how to run that program on the micro:bit.

1. What is the micro:bit?

The micro:bit is a small, programmable, microcontroller designed for use in computer science education. Basically, it's a piece of hardware that you can write your own code for. It comes with several built-in sensors, such as a temperature sensor, a light sensor, an accelerometer, and more. The data from these sensors can easily be accessed by the code you write, which allows you to create powerful interactive systems.

2. Writing your first micro:bit program

Below you can see a little window with some example micro:bit code. You'll notice that the code is written using blocks as opposed to traditional text-based programs you may be used to seeing. You can program the micro:bit using both kinds of programs, but in these tutorials we will mostly focus on the "blocks-based" way. Now, click the upper right icon to open this code example in a new window.


You are now using what is called the MakeCode editor which is one of several applications you can use to write micro:bit programs. You'll notice this example program only has two blocks: on start and show icon. Notice how the show icon block is nested inside of the on start block. The on start block allows you to do something on your micro:bit as soon as it turns on. In other words, when this program runs on the micro:bit we will see a giraffe appear on the LED screen.

3. Saving your code to your computer

When we write code for the micro:bit we end up with a file that has a .hex extension. This is just the filetype that the micro:bit expects. So, now that we have finished writing code using the online MakeCode editor, we need to save that .hex file. You can do this by clicking either the download or save button. Either works. Now you'll have a file named something like microbit-showiconexample.hex in your computer's Downloads folder. The next step will explain how to get this file onto the actual micro:bit device itself.

3. Running your code on the micro:bit

In order to transfer the hex file to your micro:bit you will have to connect your micro:bit to your computer using a microUSB cable. It will show up on your computer as if you just inserted a flash drive. Now, just drag the hex file you downloaded in step 3 and drag it onto the micro:bit. You should see a little yellow LED flashing to indicate that the program is being loaded onto the micro:bit. When that finishes you will see a giraffe show up on the LED screen!
